# Employee Leave Management System 🏢

A real-world AI-powered leave management system built using **Claude AI** and **MCP (Model Context Protocol)** via Claude Desktop.

Built on Day 3 of my AI development journey.

---

## What it does

- **Leave Request** — Employee submits a leave request with dates and reason
- **Approval System** — Manager reviews and approves or rejects requests
- **Claude AI** handles the logic, responses, and decision suggestions via MCP

---

## How it works

1. Employee describes their leave request in natural language to Claude
2. Claude (via MCP) processes the request and formats it properly
3. Manager gets a structured summary and can approve or reject
4. Claude responds with confirmation and updates the status

---

## Tech used

- Claude Desktop (Claude AI interface)
- MCP (Model Context Protocol) — connects Claude to the system
- JavaScript (for logic and array operations)

---

## What I learned building this

- How MCP connects Claude to real tools
- How Claude can handle structured data like leave requests
- How to use natural language to power a business workflow
- Arrays, filter, map, reduce in a real use case
